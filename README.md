# Western Blot Fold Change Analyzer

A static browser app for comparing sample protein band signal against a loading control.

## Analysis

1. Read one or more sample sheets and either one shared loading-control sheet or one control sheet per sample.
2. Identify name/lane and signal columns. A `Name` column is preferred by default when present.
3. Convert each signal to percent of the selected normalization lane.
4. Divide sample percent by loading-control percent for the same lane.
5. Render the adjusted fold change as a bar chart and table.

In shared-control mode, choose how many sample files use the same control file. Each sample gets its own result tab.

In comparison mode, each sample file has a matching control file. The app calculates each pair first, then lets you edit lane labels so matching labels can be compared. The final comparison graph only includes lane labels present in every sample/control pair. It can render grouped bars or average bars with individual points, and the same interleaved/block/custom grouping controls can split those comparison graphs into multiple grouped charts.

The default normalization lane is the first lane, but the selector is already wired so a different lane can be chosen. After analysis, graph labels can be edited without changing the source lane matching. Grouped graphs can be generated by interleaving every N lanes, splitting into blocks of N lanes, or manually selecting custom lane groups.

## Run

Open `index.html` directly in a browser, or serve the folder locally:

```sh
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.
